# 🔐 Security Playground

> Learn Cybersecurity Through Hands-On Practice

🌐 **Live Demo:** https://security-playground.onrender.com/

Security Playground is an interactive cybersecurity learning platform that combines educational content, realistic web security labs, and CTF-style challenges to help users develop practical cybersecurity skills in a safe environment.

Built for students, aspiring security professionals, bug bounty beginners, and anyone interested in understanding how web vulnerabilities work in the real world.

## 🚀 Why I Built This

As a cybersecurity student, I wanted a platform that goes beyond theory and allows users to learn by doing. Security Playground was created to provide an engaging, hands-on environment where learners can explore common web vulnerabilities, understand attack techniques, and practice in a safe setting.

This project also served as an experiment in AI-assisted software development, demonstrating how modern development workflows can accelerate the creation of practical security tools.

## ✨ Key Features

* Interactive cybersecurity labs
* Backend-verified CTF challenges
* Real SQL Injection lab
* Real XSS lab
* Secure server-side flag validation
* XP and progress tracking
* Modern responsive UI
* Session-based challenge progression
* AI-assisted ("vibe coded") development workflow

## 🧪 Real Labs Included

### Database Flag Extractor

A real SQL Injection challenge that allows users to practice:

* Database enumeration
* UNION-based SQL injection
* Data extraction techniques

### Admin Cookie Stealer

A real Stored XSS challenge covering:

* Stored XSS attacks
* Payload crafting
* Session hijacking concepts
* Browser exploitation fundamentals

## ⚡ Built Through Vibe Coding

This project was built using an AI-assisted development workflow often referred to as **vibe coding**.

Rather than manually writing every line from scratch, development involved collaborating with AI tools to rapidly prototype features, generate code, explore ideas, debug issues, and accelerate implementation. The final platform was then refined through testing, customization, deployment, and security-focused improvements.

This repository demonstrates how AI-assisted development can be combined with cybersecurity knowledge to create practical educational software.

```bash
cd security-playground

npm install
npm run build:flags
npm start
```
